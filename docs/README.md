## 📖 项目概述

**Shutong（书童）** 是一个 Android 应用/工程样板项目，  
项目目标是围绕「学习支持 / 内容陪伴 / 能力扩展」等方向，  
探索可持续演进的应用架构与工程组织方式。

项目强调：

- 清晰的工程边界
- 可演进的模块化设计
- 面向长期维护与扩展的结构规划

---

## 🎯 项目目标

- 构建一个结构清晰、职责明确的 Android 工程
- 支持后续在学习、记录、辅助与 AI 能力方向上的功能扩展
- 作为可复用的工程基础，为后续相关应用提供参考

---

## 🏗️ 架构与工程方向

> 具体方案将随项目推进逐步完善

- Android 平台
- Java / Kotlin 混合开发（逐步向 Kotlin 倾斜）
- Jetpack 组件（按需引入）
- 模块化工程结构
- 关注工程可维护性、可扩展性与可理解性
